# 🐾 Pet & Mount Search (WoW 3.3.5 Addon)

A simple and lightweight addon for **World of Warcraft 3.3.5** that lets you quickly search and summon your pets and mounts.

---

## ✨ Features

* 🔍 Real-time search as you type
* 🐾 Filter between **Pets** and **Mounts**
* 🖱️ Click icons to instantly summon
* ⚡ Fast performance using cached data
* 🎯 Clean and minimal UI

## 🚀 Usage

1. Type: /ps
2. Select what you want to search for
3. Search for a pet or mount
4. Click an icon to summon it
